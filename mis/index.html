MIS

管理信息系统

用于 MIS 课堂展示的管理信息系统演示。

🚧 建设中

这里将放置一个用于课堂演示的管理信息系统（增删改查、报表、权限等）。由于需要后端与数据库， 它无法在纯静态站点中实现，因此本页目前为占位。

未来动态实现的技术预留

本站当前为纯静态输出。要落地真正的 MIS，按以下路径升级，且不影响其余页面继续静态化：

  1. 给 Astro 接一个 SSR 适配器（@astrojs/node 或 @astrojs/cloudflare），整体切换为 hybrid 输出。
  2. 仅本路由组 /mis/** 标注 export const prerender = false，按需服务端渲染；其余页面保持预渲染。
  3. 交互部分用 islands（React/Vue/Svelte 组件，client:* 指令）按需注水，数据走服务端 endpoint 或外部 API。
  4. 数据与鉴权后端单独部署（或用边缘函数 + 托管数据库）。

路由与导航入口已预留，届时只需在此目录补充动态页面与组件。
